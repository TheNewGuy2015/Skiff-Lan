Learn More

This website was created to help students use the LAN and to provide more information about the LAN.
This Lan is availble to anyone that has a Skiff Annex username and password. It was created to allow students access to many different operating systems on any computer anywhere in the world as long as you have internet access. It was partially completed in 2015, allowing students to SSH into the server. In, 2016 the VPN was finalized giveing access to all of the Virtual Machines (VMs). We are currently working on a DNS Server (Domain Name Services Server) to allow student to connect to computers by a name insted of an IP (Internet Protocol) Address.
